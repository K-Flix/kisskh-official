# **App Name**: StreamVerse

## Core Features:

- Browse Movies and TV Shows: Allow users to browse movies and TV shows from the TMDB API.
- Search Functionality: Enable users to search for specific titles or genres.
- Detail Pages: Provide detailed information for each title, including synopsis, cast, and ratings. Also, suggest similar shows using AI tool
- Personalized Recommendations: Recommend movies and TV shows based on viewing history (if implemented).
- Watchlist: Allow users to add movies and TV shows to a watchlist.
- Category Grouping: Movies are categorized by genres to allow user to find easily

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey a sense of sophistication and drama.
- Background color: Very dark gray (#121212) for a cinematic viewing experience.
- Accent color: Teal (#00BCD4) to provide contrast and highlight key elements.
- Body and headline font: 'Inter', a sans-serif, for a clean, modern, and easily readable interface.
- Use simple, outlined icons for navigation and controls.
- Grid-based layout for displaying movies and TV shows.
- Smooth transitions and loading animations to enhance user experience.
